# Wizard Powers

## Vision
A toolkit that hooks AI agents into real-world APIs — things with genuine physical, legal, or financial impact that can be triggered entirely digitally. Think: form an LLC, send certified mail, file a tax document, bind an insurance policy, buy a domain — all from a prompt.

## Goal
Map the full landscape of "wizard power" APIs (real-world outcomes reachable via API), then build agent-ready integrations for the most impactful ones.

## Phase
Currently in **research** — landscape mapped, integration work not started. See `docs/research-landscape.md`.
