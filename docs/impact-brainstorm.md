# Wizard Powers — Real-World Impact Brainstorm

Organized by what actually happens in the world, not how it's done.

## Legal & Government

- Form a legal business entity (LLC, C-Corp, nonprofit) in any US state
- Register a DBA ("doing business as") name
- Apply for a 501(c)(3) nonprofit designation
- File for trademark protection on a brand name or logo
- Record a copyright registration
- File a DMCA takedown notice against infringing content
- Send a legally valid demand letter to collect a debt
- File a lien against someone's property for unpaid work
- File a UCC-1 financing statement to secure a loan
- Open a bankruptcy case
- Submit a small claims court filing
- File a motion in federal court
- Apply for a restraining order (in jurisdictions with e-filing)
- File a FOIA (Freedom of Information Act) request with any federal agency
- Submit public comment on a proposed federal regulation (via regulations.gov)
- Report suspected tax fraud to the IRS (Form 3949-A)
- File a whistleblower complaint with the SEC
- Submit a complaint to the CFPB against a financial institution
- Place a legal notice in a newspaper of record (required for some LLC formations)
- Apply for a business license in a municipality
- Register for a sales tax permit in a state
- Get an EIN (federal Employer Identification Number) for a business
- Apply for a liquor license (initial application paperwork)
- Apply for an FCC license (amateur radio, etc.)
